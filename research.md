## 2. 사용자들이 공통적으로 불만을 느끼는 점

### 커스터마이징 부족
- **75%의 사용자**가 커스터마이징이 부족하다고 느낌
- 팀 피드백 수집 기능에서 **67%가 문제 경험**

### 기술적 문제
- 버그/글리치로 인한 네비게이션 및 리포트 생성 문제
- 후보자 프로필에 파일 첨부 시 **90%가 문제 경험**
- 잡보드 전환 시 로딩 지연, 세션 만료
- AI 도구 성능 기대 이하

### AI/파싱 정확도
- "CV 파싱이 정확하지 않음, AI 피드백도 부정확"
- 이력서 업로드 시 정보 추출 실패
- 중복 후보자 프로필 빈번 발생

### 고객 지원
- 느린 응답 시간
- 라이브 서포트 연결 어려움
- 이메일 지원에만 의존

### 연동(Integration) 문제
- 연동 시 추가 비용 발생
- 특정 도구와의 호환성 문제
- HR, 재무, 영업, 운영 시스템 간 통합 부족

### 리포팅 한계
- 기본적인 리포트만 제공
- AI 분석, 복잡한 리포팅 기능 부족

### 가격
- 소규모 비즈니스에게 높은 가격
- 개인화된 청구 옵션 부재

### 사용자 경험
- 지원자 입장에서 경험이 단조롭고 매력 없음
- 학습 곡선 존재, 추가 교육 필요

## 3. 아직 아무도 해결 못한 문제

### 3-1. AI ROI 및 실행 격차
- CEO의 10% 미만만 AI 투자 대비 수익 확신
- HR 리더 92%가 AI 구현 참여하지만, **21%만 AI 전략 결정에 관여**
- AI 도입은 빠르지만 실제 효과 측정은 미흡

### 3-2. 데이터 프라이버시 & 컴플라이언스
- HR Tech 스타트업 62%가 민감한 직원 데이터를 클라우드에 저장
- **39%가 GDPR, CCPA 등으로 인해 글로벌 확장 지연**
- 컴플라이언스 프레임워크 구축 비용이 운영 예산의 평균 11%
- 비즈니스 리더 56%가 데이터 보안/직원 데이터 프라이버시를 최우선 기술 장애물로 인식

### 3-3. 스킬 갭 & 업스킬링
- 2030년까지 **59%의 인력이 재교육 필요**
- 2030년까지 **현재 스킬의 39%가 쓸모없어질 것**
- 하지만 59%의 근로자가 적절한 교육을 받지 못할 것
- AI 특화 업스킬링이 높은 영향력을 가진다고 54%가 응답했지만, **실제 구현은 1%만**

### 3-4. 중간 관리자 & 리더십 파이프라인 위기
- 82%의 보드/CEO가 AI로 인해 향후 3년 내 최대 20% 인력 감축 예상
- 중간 관리자와 엔트리 레벨 제거가 **미래 리더십 파이프라인 파괴**
- "스프레드시트에서는 좋아 보이지만, 미래 리더십 벤치를 조용히 파괴"

### 3-5. 일선 근로자(Frontline Worker) 경험
- 전 세계 인력의 **80% 이상이 일선 근로자**
- 소매, 호스피탈리티 등 일선 직원 경험 플랫폼(FEXP) 수요 증가
- 하지만 대부분의 HR Tech는 사무직 중심

### 3-6. 윤리적 AI & 거버넌스
- AI가 고용 결정에 영향 → 규제 기관 대응 증가
- 컴플라이언스가 기술 선택의 핵심 요소로 부상
- 벤더 차별화의 주요 요인

## 4. 진입하기 좋은 틈새시장

### 4-1. SMB(중소기업) 채용 시장 ⭐ 추천

**기회 요인:**
- SMB 채용 담당자의 **50%가 급여 기대치 충족 어려움**
- **40%+가 필요 스킬 보유 후보자 찾기 어려움**
- **40% 가까이가 느린 채용 프로세스로 우수 후보자 놓침**
- **40%+가 회사 문화 적합성 평가 어려움**
- 전담 TA팀 없이 린(Lean) HR팀으로 운영

**차별화 포인트:**
- 대기업은 RTO(사무실 복귀) 강요 → SMB는 하이브리드/리모트 제공 가능
- SMB는 채용 결정이 빠름 (이해관계자 적음, 관료주의 적음)
- 스킬 기반 채용에서 SMB가 유연성 우위 (부트캠프 졸업생이 72% 더 우수한 성과)
- 패시브 후보자 소싱에서 기회 (대기업이 놓치는 인재)

### 4-2. 한국 스타트업/중소기업 채용 ⭐⭐ 강력 추천

**시장 현황:**
- 2022년 5월: IT/플랫폼 기업 신규 채용 공고 8,498건 → 2025년 6월: 4,872건 (약 43% 감소)
- 2022년 5월: 채용 성공 1,596건 → 2025년 6월: 773건 (약 52% 감소)
- 112개 VC 투자 스타트업이 2025년에 30% 이상 인력 감축

**기회 요인:**
- 대기업 만족도: 2022년 50.4% → 2024년 65% (격차 확대)
- 스타트업이 인재 경쟁에서 대기업에 밀리는 중
- **스타트업/중소기업 전용 채용 브랜딩 지원 수요**
- 원티드가 IT/스타트업 특화로 성공 → 틈새 특화 전략 유효

### 4-3. 자동화된 지원자 커뮤니케이션

**기회 요인:**
- 지원자 경험(Candidate Experience)이 단조롭다는 불만 다수
- 상태 변경 시 자동 이메일이 기본이지만, 개인화/타이밍 최적화 부족
- 지원 후 "블랙홀"에 빠진 느낌이라는 지원자 불만

**차별화 포인트:**
- 상태별 맞춤 이메일 템플릿 (이미 Antigravity에 구현됨)
- 발송 로그 및 분석 (이미 계획됨)
- 지원자 입장의 투명한 프로세스 공개

### 4-4. 심플한 ATS (No-Frills ATS)

**기회 요인:**
- 기존 ATS는 기능이 많아 학습 곡선 존재
- SMB는 복잡한 기능보다 **"바로 쓸 수 있는" 단순함** 원함
- Breezy HR, Manatal이 이 시장에서 성공

**차별화 포인트:**
- 5분 내 설정 완료
- 직관적인 칸반 보드 파이프라인
- 필수 기능만 제공 (공고, 지원, 상태 관리, 이메일)

### 4-5. 채용 브랜딩 + ATS 통합

**기회 요인:**
- 회사 소개/문화 페이지와 채용 시스템이 분리된 경우 많음
- 지원자가 회사 문화를 이해하고 지원하면 적합도 높아짐
- Antigravity가 이미 company/culture 페이지 보유 → 자연스러운 통합 가능

## 5. Antigravity 프로젝트 포지셔닝 제안

### 타겟 시장
> **한국 스타트업/중소기업**을 위한 **심플하고 브랜딩 중심**의 채용 플랫폼

### 핵심 차별점

| 기존 서비스 문제 | Antigravity 해결책 |
|-----------------|-------------------|
| 복잡한 설정, 높은 학습 곡선 | 5분 내 설정, 직관적 UI |
| 지원자 경험 단조로움 | 자동 이메일 + 투명한 상태 공개 |
| 채용 브랜딩 별도 필요 | 회사/문화 페이지 내장 |
| 고가의 엔터프라이즈 가격 | SMB 친화적 가격 |
| AI 기능은 있지만 ROI 불명확 | 필수 기능에 집중, 명확한 가치 |

### 우선순위 기능

1. **채용 공고 관리** - 등록/수정/공개/비공개
2. **지원자 관리** - 칸반 보드 형태 상태 관리
3. **자동 이메일** - 지원완료/면접/합격/불합격
4. **이메일 로그** - 발송 이력 및 분석
5. **채용 브랜딩** - 회사/문화 페이지 통합
